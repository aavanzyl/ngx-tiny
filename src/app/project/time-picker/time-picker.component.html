ngx-time-picker

Installation

To install this library, run:

npm install @ngx-tiny/ngx-time-picker --save

and then add NgxTimePickerModule to you AppModule

* Note that the default locale for the ngx-time-picker is en-US. We would recommend this gets specified for the target local.

Usage

Styling

You can customize the styling of the ngx-time-picker by including the style file and setting the required variables

Include the ./node_modules/ngx-time-picker/theme.scss as an import in your styles.scss file.

API

Input

Input specification on ngx-time-picker component.

Attribute Default Required Specification
value new Date() optional

Type: Date

Current value of time picker.
options @see Options optional

Type: TimePickerOptions

Options of the Time Picker


Output

Output specification on ngx-time-picker component.

Attribute Required Specification
valueChange optional

Type: EventEmitter

Date or DateRange


Options

Available Options on ngx-time-picker component.

@see DatePickerOptions in ngx-time-picker

Attribute Default Specification
closeOnClickOutside true

Type: boolean

Close Date Picker when clicked outside of container.
locale en-US

Type: object

Specify the Date-fns locale to be used.

If you don't specify a locale in options, default locale is en-US.

See https://date-fns.org/v2.8.1/docs/Locale
displayFormat MMM d, yyyy

Type: date-fns format

Format of display value of a date.

See https://date-fns.org/docs/format
hoursStep 1

Type: number

Number of hours to step on click
minutesStep 1

Type: number

Number of minutes to step on click
military false

Type: boolean

Choose to use 24h display or 12h display
