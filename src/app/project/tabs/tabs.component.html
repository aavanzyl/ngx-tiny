ngx-tabs

Installation

To install this library, run:

npm install @ngx-tiny/tabs --save

and then add NgxTabsModule to you AppModule

Styling

You can customize the styling of the ngx-tabs by including the style file and setting the required variables

Load it from ./node_modules/ngx-tabs/theme.scss via package bundler like Angular CLI, if you're using one.

or

Include the ./node_modules/ngx-tabs/theme.scss as an import in your styles.scss file.

API

ngx-tabs

Attribute Default Required Specification
active false optional

Type: Boolean

Set the selected tab as active
activeTabIndex 0 optional

Type: Number

Set a tab index as active

ngx-tab-routes

Attribute Default Required Specification
tabs [] required

Type: Object with fields name and path

Array of tab router object listing the required routes.
