ngx-quill-editor


Implementation of the Quill Editor that lazy loads the required JS and Styling only when required.


Installation

To install this library, run:

npm install @ngx-tiny/quill-editor --save

and then add NgxQuillEditorModule to you AppModule

Lazy loads the following files when initializing the editor:

  • cdn.quilljs.com/1.3.6/quill.min.js
  • cdn.quilljs.com/1.3.6/quill.snow.css

Usage

API

Input

Output specification on ngx-quill-editor component.

Attribute Default Required Specification
options optional

Type: object

@see https://quilljs.com/docs/configuration/

Output

Output specification on ngx-multi-select component.

Attribute Required Specification
blurred optional

Type: EventEmitter

Emits even on input blur
focused optional

Type: EventEmitter

Emits event on input focus
ready optional

Type: EventEmitter

Emits event on input ready
changed optional

Type: EventEmitter

Emits event on input changed with editor, html & text as output in object form.
