ngx-switch-input

Installation

To install this library, run:

npm install @ngx-tiny/switch-input --save

and then add NgxSwitchInputModule to you AppModule

Usage

Styling

You can customize the styling of the ngx-switch-input by including the style file and setting the required variables

Load it from ./node_modules/ngx-switch-input/theme.scss via package bundler like Angular CLI, if you're using one.

or

Include the ./node_modules/ngx-switch-input/theme.scss as an import in your styles.scss file.

API

Input

Output specification on ngx-switch-input component.

Attribute Default Required Specification
value null optional

Type: boolean

Value of the switch, use when using as a standalone instance.

Output

Output specification on ngx-switch-input component.

Attribute Required Specification
valueChange optional

Type: EventEmitter

Emits a boolean if the switch value changes.

Properties

Output specification on ngx-switch-input component.

Attribute Default Required Specification
label null optional

Type: string

Optional Label for the switch, displayed on the left of the switch.
