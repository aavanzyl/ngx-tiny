ngx-multi-select

Angular modern multi selection dropdown.

  • Angular 8 implementation
  • Works with SSR
  • Dynamic filtering of items
  • Supports Keyboard navigation (arrow, space and enter keys)
  • Works alone or with angular forms
  • No Dependencies required
  • String or Object Array as options
  • Themeable to own requirements

Installation

To install this library, run:

npm install @ngx-tiny/multi-select --save

and then add NgxMultiSelectModule to you AppModule

API

Input

Input specification on ngx-multi-select component.

Attribute Default Required Specification
selected [] optional

Type: MultiSelectOption or string

Array of selected options. Only required when not using forms.
options [] required

Type: MultiSelectOption or string

Array of options to select from
config optional

Type: MultiSelectConfiguration

Configuration for the dropdown


Output

Output specification on ngx-multi-select component.

Attribute Required Specification
valueChange optional

Type: EventEmitter

MultiSelectOption or string


Properties

Available Properties on ngx-multi-select

Attribute Default Specification
placeholder 'Select Option'

Type: string

String placed when no options is selected.
deliminator ', '

Type: string

Deliminator between selected options.
searchPlaceholder Search...

Type: string

Placeholder text for search input field
searchIconClass default-icon

Type: string

Option to replace with font-awesome or other icon lib than one built in
searchBox SearchBoxOption.DYNAMIC

Type: SearchBoxOption

Choose when to display the searchbox of the dropdown. DYNAMIC looks at searchBoxDynamicLimit to determine if it should be visible.
searchBoxDynamicLimit 5

Type: number

Breaking point at which the searchbox should show or hide based on number of options.
