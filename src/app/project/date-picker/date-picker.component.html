ngx-date-picker


Installation

To install this library, run:

npm install @ngx-tiny/ngx-date-picker --save

and then add NgxDatePickerModule to you AppModule

* Note that the default locale for the ngx-date-picker is en-US. We would recommend this gets specified for the target local.

Usage

Styling

You can costomize the styling of the ngx-date-picker by inlcuding the style file and setting the required variables

Load it from ./node_modules/ngx-date-picker/theme.scss via package bundler like Angular CLI, if you're using one.

or

Include the ./node_modules/ngx-date-picker/theme.scss as an import in your styles.scss file.

Attribute Default Required Specification
value new Date() optional

Type: Date or DateRange

Current value of date picker.

Output

Output specification on ngx-date-picker component.

Attribute Required Specification
valueChange optional

Type: EventEmitter

Date or DateRange

Options

Available Options on ngx-date-picker component.

@see DatePickerOptions in ngx-date-picker

Attribute Default Specification
closeOnClickOutside true

Type: boolean

Close Date Picker when clicked outside of container.
closeOnSelection true

Type: boolean

Close Date Picker value is selected or End Value is selected in range.
locale en-US

Type: object

Specify the Date-fns locale to be used.

If you don't specify a locale in options, default locale is en-US.

See https://date-fns.org/v2.8.1/docs/Locale
includeDays all

Type: string

Options 'none' | 'previous-month' | 'next-month' | 'all'

Specify if pre/post days need to be shown in the picked.
includeNextMonthsFirstFullWeek false

Type: boolean

Ability to include a full week from the next month.

displayFormat MMM d, yyyy

Type: date-fns format

Format of display value of a date.

See https://date-fns.org/docs/format
barTitleFormat MMMM YYYY

Type: date-fns format

Format the date displayed on the top bar of the date picker.

See https://date-fns.org/docs/format
dayNamesFormat ddd

Type: date-fns format

Format the Names of the days shown on the date picker

See https://date-fns.org/docs/format
barTitleIfEmpty 'Click to select a date'

Type: date-fns format

Title of the text on the top bar when no date specified.
useEmptyBarTitle true

Type: date-fns format

If false, barTitleIfEmpty will be disregarded and a date will always be shown.
selectRange false

Type: boolean

Specify if the Date Picker is single or range select
rangeSeparator -

Type: string

Separator between range date on display.
firstCalendarDay 0

Type: number

Start of the calendar day, 0 = Sunday (default), 1 = Monday, ..
minYear 1900

Type: number

Year Selection start
maxYear 2050

Type: number

Year Selection end
minDate undefined

Type: Date

The minimal date allowed for selection.
maxDate undefined

Type: Date

The maximum date allowed for selection.
