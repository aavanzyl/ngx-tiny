Getting Started

To get started find the component you would like to use in the navigation. There is no single install for @ngx-tiny components and libraries. Each component or library is a stand alone npm package. Each component is documented with how to install and use in your Angular application.

Use the side menu to navigate to the individual components, check out the demos and install the ones you want to use.
